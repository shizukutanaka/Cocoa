# Otedama システム - 利用者ガイド

Otedamaは、VRChatなどのメタバースプラットフォームで使用するアバター、プリセット、アクセサリーを安全に管理・共有できるシステムです。このガイドでは、Otedamaの基本的な使い方から高度な機能までを、利用者目線でわかりやすく説明します。

## システム概要

Cocoaは、以下の特徴を持つプロフェッショナルなアバター管理システムです：

### 主要機能
- **アバター管理**: 複数のアバターを効率的に整理・管理
- **プリセット管理**: パラメータセットを保存・共有
- **クラウド同期**: 複数のデバイス間で設定を同期
- **セキュリティ**: 軍事レベルのセキュリティ機能
- **バックアップ**: 自動バックアップと復元機能
- **パフォーマンス**: 高速処理と最適化機能

### 対象ユーザー
- **個人ユーザー**: 趣味でVRChatを楽しむ方
- **クリエイター**: アバターを作成・販売する方
- **企業**: バーチャルオフィスやイベントで使用する方
- **機関**: 教育・研究機関での利用

## インストールとセットアップ

### システム要件

#### 最小要件
- **OS**: Windows 10 64-bit / macOS 10.15 / Ubuntu 20.04
- **CPU**: Intel Core i5 または同等
- **メモリ**: 8GB RAM
- **ストレージ**: 10GB 以上の空き容量
- **ネットワーク**: ブロードバンドインターネット接続

#### 推奨要件
- **CPU**: Intel Core i7 または同等
- **メモリ**: 16GB RAM
- **ストレージ**: SSD 500GB
- **GPU**: NVIDIA GTX 1060 または同等（VR機能使用時）

### インストール手順

#### Windowsの場合

1. **インストーラーを入手**
   - 社内配布サーバーまたは公式ダウンロードページから `cocoa-setup.exe` を取得します。

2. **インストーラーを実行**
   ```bash
   # ダウンロードしたファイルをダブルクリック
   cocoa-setup.exe
   ```

3. **インストールウィザードに従う**
   - インストール先を選択（デフォルト推奨）
   - 追加コンポーネントの選択
   - インストール完了まで待つ

#### macOSの場合

1. **DMGファイルを入手**
   - 社内配布サーバーまたは公式ダウンロードページから `Cocoa.dmg` を取得します。

2. **DMGファイルを開く**
   ```bash
   hdiutil attach Cocoa.dmg
   ```

3. **アプリケーションをインストール**
   - CocoaアイコンをApplicationsフォルダにドラッグ
   - DMGファイルをアンマウント

#### Linuxの場合

1. **AppImageを入手**
   - 社内配布サーバーまたは公式ダウンロードページから `Cocoa.AppImage` を取得します。
   ```bash
   chmod +x Cocoa.AppImage
   ```

2. **実行**
   ```bash
   ./Cocoa.AppImage
   ```

### 初回起動とセットアップ

1. **アプリケーションを起動**
   - デスクトップアイコンまたはスタートメニューから起動

2. **言語の選択**
   - 日本語を選択（他の言語も利用可能）

3. **ライセンス同意**
   - 利用規約を確認し、同意する

4. **初期設定**
   - ユーザー名を設定
   - データ保存場所を選択
   - クラウド同期を設定（任意）

5. **完了**
   - セットアップ完了メッセージが表示されたら完了

## 基本的な使い方

### 初めての起動

Cocoaを初めて起動すると、以下のような画面が表示されます：

```
┌─────────────────────────────────────────┐
│           Cocoa アバター管理           │
├─────────────────────────────────────────┤
│                                         │
│  [1] 新しいアバターを作成               │
│  [2] 既存のアバターを読み込み           │
│  [3] プリセットを管理                   │
│  [4] 設定を開く                         │
│  [5] ヘルプを表示                       │
│                                         │
│  番号を選択してください (1-5):        │
└─────────────────────────────────────────┘
```

### 基本操作の流れ

#### 1. アバターの作成
1. メインメニューから「1」を選択
2. アバターテンプレートを選択
3. 基本パラメータを設定（身長、体型など）
4. 詳細設定で微調整
5. 保存して完了

#### 2. アバターの読み込み
1. メインメニューから「2」を選択
2. VRChatからエクスポートしたファイルを指定
3. 自動的に解析・最適化
4. 管理画面に追加

#### 3. プリセットの管理
1. メインメニューから「3」を選択
2. 現在の設定をプリセットとして保存
3. 他のプリセットを読み込み適用
4. プリセットの編集・削除

## アバター管理

### アバター一覧の見方

アバター管理画面では、以下の情報が表示されます：

```
┌─────────────────────────────────────────────────────────────┐
│                    アバター一覧                             │
├─────────────────────────────────────────────────────────────┤
│ 名前              │ タイプ  │ サイズ │ 更新日     │ 状態    │
├─────────────────────────────────────────────────────────────┤
│ ビジネスアバター  │ フル    │ 45MB  │ 2024-01-15 │ アクティブ│
│ カジュアルアバター│ シンプル│ 12MB  │ 2024-01-14 │ アクティブ│
│ パーティーアバター│ フル    │ 38MB  │ 2024-01-13 │ バックアップ│
└─────────────────────────────────────────────────────────────┘
```

### アバターの詳細管理

#### アバターの編集
1. アバター一覧から編集したいアバターを選択
2. 「編集」ボタンをクリック
3. パラメータタブで調整
4. 変更を保存

#### アバターの削除
1. アバター一覧から削除したいアバターを選択
2. 「削除」ボタンをクリック
3. 確認ダイアログで「はい」を選択
4. 30日間はゴミ箱から復元可能

#### アバターのエクスポート
1. エクスポートしたいアバターを選択
2. 「エクスポート」ボタンをクリック
3. エクスポート形式を選択（VRChat用など）
4. 保存先を指定

## プリセット管理

### プリセットとは

プリセットとは、アバターのパラメータ設定を保存したものです。例えば：
- 「ビジネスモード」：仕事用の設定
- 「パーティーモード」：イベント用の設定
- 「カジュアルモード」：日常用の設定

### プリセットの作成

1. アバターを調整して好みの設定にする
2. 「プリセット」メニューを開く
3. 「新規プリセットを作成」を選択
4. プリセット名を入力
5. 保存するパラメータを選択
6. 「保存」ボタンをクリック

### プリセットの活用

#### プリセットの適用
1. プリセット一覧から適用したいプリセットを選択
2. 「適用」ボタンをクリック
3. アバターに設定が反映される

#### プリセットの共有
1. 共有したいプリセットを選択
2. 「共有」ボタンをクリック
3. 共有方法を選択（リンク、ファイルなど）
4. 相手に送信

## 高度な機能

### クラウド同期

#### セットアップ
1. 「設定」メニューを開く
2. 「クラウド同期」を選択
3. アカウントを作成またはログイン
4. 同期するデータを選択
5. 「同期を開始」をクリック

#### 同期の管理
- **自動同期**: 変更を自動的にクラウドに保存
- **手動同期**: 必要な時だけ同期
- **選択的同期**: 重要なデータのみ同期

### バックアップ機能

#### 自動バックアップ
- 毎日午前2時に自動実行
- 30日分のバックアップを保持
- 重要なデータは暗号化して保存

#### 手動バックアップ
1. 「バックアップ」メニューを開く
2. 「今すぐバックアップ」を選択
3. バックアップ範囲を選択
4. 実行して完了

#### 復元
1. 「復元」メニューを開く
2. 復元ポイントを選択
3. 復元範囲を選択
4. 「復元実行」をクリック

### セキュリティ機能

#### 二段階認証
1. 「設定」→「セキュリティ」を開く
2. 「二段階認証」を有効化
3. QRコードをスキャン
4. 確認コードを入力

#### データ暗号化
- すべてのデータはAES256で暗号化
- クラウド同期も暗号化
- バックアップデータも保護

### パフォーマンス最適化

#### 自動最適化
- 起動時に自動的に最適化を実行
- 不要なキャッシュを削除
- メモリ使用量を最適化

#### 手動最適化
1. 「設定」→「パフォーマンス」を開く
2. 「最適化を実行」をクリック
3. 完了まで待つ

## トラブルシューティング

### よくある問題と解決方法

#### アプリケーションが起動しない
**症状**: Cocoaが起動せず、エラーメッセージが表示される

**解決方法**:
1. システム要件を確認
2. セキュリティソフトの例外設定を確認
3. インストールを修復
4. ログファイルを確認（`logs/cocoa.log`）

#### アバターが正しく表示されない
**症状**: 読み込んだアバターが正しく表示されない

**解決方法**:
1. ファイルが破損していないか確認
2. VRChatのバージョンと互換性があるか確認
3. 再インポートを試す
4. サポートに連絡

#### 同期が上手くいかない
**症状**: クラウド同期がエラーになる

**解決方法**:
1. ネットワーク接続を確認
2. 同期設定を見直す
3. キャッシュをクリア
4. アカウント設定を確認

#### パフォーマンスが悪い
**症状**: 動作が重い、応答が遅い

**解決方法**:
1. パフォーマンス最適化を実行
2. 不要なアバターを削除
3. システムリソースを確認
4. 設定でパフォーマンスモードを有効化

### ログファイルの確認

トラブルシューティング時は、ログファイルを確認してください：

```
logs/
├── cocoa.log          # メインアプリケーションのログ
├── security.log       # セキュリティ関連のログ
├── performance.log    # パフォーマンス関連のログ
└── backup.log         # バックアップ関連のログ
```

### サポートへの連絡

問題が解決しない場合は、以下の方法でサポートに連絡してください：

1. **ヘルプメニュー**: アプリケーション内のヘルプから
2. **公式フォーラム**: 導入環境で指定されたサポート掲示板を利用してください。
3. **メールサポート**: 運用担当が案内するサポートメールアドレスに連絡してください。
4. **チャットサポート**: アプリケーション内のチャットボタンから連絡してください。

サポート連絡時は、以下の情報を必ず含めてください：
- バージョン情報
- エラーメッセージ
- 実行した操作
- システム情報（OS、メモリなど）

## よくある質問

### Q: Cocoaは無料ですか？
A: 基本機能は無料で利用できます。有料プランで高度な機能が利用可能です。

### Q: 複数のデバイスで使用できますか？
A: はい、クラウド同期機能により、複数のデバイス間でデータを共有できます。

### Q: データは安全ですか？
A: はい、軍事レベルのセキュリティで保護されています。すべてのデータは暗号化されます。

### Q: VRChat以外のアバターも管理できますか？
A: はい、汎用的なアバターフォーマットに対応しています。

### Q: バックアップはどのくらいの期間保持されますか？
A: デフォルトで30日間保持されます。設定で変更可能です。

### Q: システム要件を満たしていない場合は？
A: 基本機能は動作する可能性がありますが、最低要件を満たすことを推奨します。

### Q: アバターの数に制限はありますか？
A: 無料プランでは100個まで、有料プランでは無制限です。

### Q: 他のユーザーとアバターを共有できますか？
A: はい、安全な共有機能により、他のユーザーとアバターを共有できます。

### Q: オフラインでも使用できますか？
A: 基本機能はオフラインで使用可能です。一部の機能（クラウド同期など）はオンラインが必要です。

### Q: アップデートはどのように行われますか？
A: 自動アップデート機能により、最新版が自動的にインストールされます。手動アップデートも可能です。
